Introduction to Computer Science

Effortlessly learn how to program and develop websites. No experience necessary.

Click here for class one!

You'll learn how to:

Build Websites

You'll make elegantly designed websites using HTML and CSS. The ability to make websites will allow you to contribute to organizations, clubs, and research projects by presenting them online.

Write Programs

Using Python, you'll learn how to write programs that process data, interact with users, and automate everyday tasks. You'll learn how to automate everyday computer tasks and contribute to computation biology / chemistry projects.

Analyze Data

You'll learn how to use modern statistical computing terms to analyze, process, and display data. This will give you an intuitive sense of the data involved in projects you're working on.
